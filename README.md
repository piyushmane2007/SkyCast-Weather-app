# 🌤 Weather App

Weather App is a simple web application built using Flask and the WeatherAPI.

The app allows users to search for any city and view its current weather information, including temperature, weather condition, and humidity through a clean and modern interface.

## Features

* Search weather by city
* Displays current temperature
* Shows current weather condition
* Displays humidity
* Real-time weather data using WeatherAPI
* Modern dark-themed user interface

## Tech Stack

* Python
* Flask
* HTML
* CSS
* Requests library
* WeatherAPI (REST API)

## What I Learned

* Flask routing and request handling
* Handling GET and POST requests
* Working with HTML forms
* Consuming REST APIs
* Making HTTP requests using the Requests library
* Understanding and working with JSON data
* Passing data from Flask to HTML using Jinja templates
* Debugging common Flask errors
* Building a complete Flask application from scratch

## Project Background

I built this project while learning Flask and REST APIs. Instead of following a complete tutorial, I focused on understanding how APIs work by building the application step by step. During this project, I learned how to send HTTP requests, process JSON responses, and display dynamic weather data on a webpage. I also used AI to improve the styling and create a more modern user interface while implementing the backend logic myself.
